Metric-Imperial Converter

Usage

Uses an API endpoint to take in user input
and return result of conversion from metric
to Imperial or vice versa.

  • gallons to liters and vice versa
  • miles to kilometers and vice versa
  • pounds to kilograms and vice versa

Example: {this site URL}/api/convert?input=5.3gal

Quick Entry

Enter numerical quantity and units to be
converted. Only one decimal and one fraction
allowed. Example input: 5.3gal
